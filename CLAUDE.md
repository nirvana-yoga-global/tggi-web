# TGGI Web — Thapovanam Grow Green Initiative

## What This Is
Public website for the One Soul • One Tree Movement, run by Nirvana Yoga Global ashram, Trivandrum, Kerala. Plant registration, monthly photo uploads, community gallery.

## Tech Stack
- React + Vite + Tailwind CSS
- Supabase (shared project with NYG Connect and NYG CRM)
- Hosted on Vercel under nirvanayogaglobal account
- GitHub: github.com/nirvana-yoga-global/tggi-web

## Brand
- Palette: cream (#faf7f0), forest green (#1f3d2b), terracotta (#c4623d)
- Serif headings (Cormorant Garamond/Playfair Display), sans-serif body (Lato/Inter)
- Tone: calm, editorial, spiritual — never corporate or NGO-dashboard feel
- Mobile-first, 50+ age accessibility (16px minimum font, 48px tap targets, high contrast)

## Pages Built
- Homepage (/) — full design with all sections, gallery preview shows non-flagged content only
- Register (/register) — plant registration with email OTP verification, duplicate checks, NYG ID generation
- Gallery (/gallery) — masonry grid showing non-flagged tggi_registrations + tggi_featured_photos
- Person Profile (/person/:phone) — individual person's garden
- Plant Profile (/plant/:id) — growth timeline with monthly updates
- My Garden (/my-garden) — phone lookup, shows all user plants with pending/flagged status, monthly update flow

## Registration Flow
- Email and phone are both mandatory fields
- Email must be verified via OTP before rest of form is shown
- OTP sent via Brevo (Edge Function: supabase/functions/send-otp/index.ts), expires in 10 minutes
- OTP verified via Edge Function: supabase/functions/verify-otp/index.ts
- On submit: duplicate email check → duplicate phone check → generate NYG ID → upload photo → insert row
- NYG ID format: NYG-G-YYYY-XXX (e.g. NYG-G-2026-001), generated by src/lib/generateNygId.js
- New registrations insert with email_verified = true, is_flagged = false

## Flag / Visibility Logic
- is_flagged = false → safe to show in gallery (default for all new registrations)
- is_flagged = true → hidden from gallery, shown as "Pending Review" badge in My Garden
- Admin sets is_flagged = true in Supabase Table Editor to hide a registration
- Gallery and GalleryPreview query: .eq('is_flagged', false)
- tggi_updates and tggi_featured_photos still use is_approved (not yet renamed)

## Edge Functions
- supabase/functions/send-otp/index.ts — POST { email }, generates + stores OTP, sends via Brevo
- supabase/functions/verify-otp/index.ts — POST { email, otp }, validates + marks used
- Both require BREVO_API_KEY secret (already set in Supabase Edge Function Secrets)
- Sender email: noreply@nirvanayogaglobal.com — must be verified in Brevo
